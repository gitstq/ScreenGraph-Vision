# Changelog

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-05-21

### ✨ 新增

- 🎉 项目初始化
- 📸 截图分析核心功能
- 🎯 四种分析模式（通用、UI、访问性、差异）
- 💾 智能缓存机制
- 🌐 多语言 README（简体中文、繁体中文、英文、日语、韩语、西班牙语）
- 🎮 交互式 TUI 界面
- 🔧 跨平台截图支持（macOS、Linux、Windows）

### 🏗️ 技术特性

- ⚡ 零依赖设计（仅使用 Python 标准库）
- 🐍 Python 3.7+ 兼容
- 📦 多种安装方式（直接运行、pip、CLI工具）
- 🐛 完整的单元测试覆盖
- 📚 详尽的文档

### 🎯 核心功能

1. **通用描述模式** - 自动分析截图内容，生成详细描述
2. **UI 元素分析模式** - 识别按钮、输入框、链接等 UI 组件
3. **可访问性评估模式** - 分析色彩对比度、文本可读性
4. **差异对比模式** - 对比两张截图的变化
